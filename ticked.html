Ticked

Ticked is a mobile productivity app for students between the ages of 16 to 24 who want to keep their study plans and to-do lists in one place and manage them on the go. As the UI designer for this student project I conducted research, created the design, and created a prototype with Adobe XD, Illustrator, Draw.io, and Prott.

My Process

Research

Analyzing existing apps provides insight into similar product problems and issues. This allows me to properly evaluate which solutions would work best and add any customizations or improvements for my project. I compared two popular productivity apps and analyzed their strengths and weaknesses.

Ideation

After analyzing the top existing apps I decided the best approach for my audience would be to create a simple to use application with minimal but strong features. There should also be no onboarding process so the app is ready to use after download. Also the app needs clear design patterns so users don’t need to guess and learn how the app works.

Wireframe

The research allowed me to flush out the most important features for the app and create a user flow diagram. The diagram helps with visualizing each step users would need to take to accomplish their tasks and give me a template for my low and mid fidelity wireframes

Design

My style guide was built on the principles of minimalism with vibrant accent colors. Because users are between the ages 16 - 24 I wanted to make sure the style was simple, engaging, and modern.

prototype

Once I worked out the user flow, wireframes, and style guide I was able to put it all together and create the final screens. These screens were then prototyped, tested, and tweaked to create the final screens.

Conclusion

I was able to accomplish designing the UI for the target audience by creating a clean and modern design. The organic feel coupled with strong accent colors really help make the design appealing and simple to read.

The user flow is simple and intuitive which allows students to start using the app quickly with little to no learning curve. This elevates the pain point of learning a new app's features after downloading.

Although I like this design I know that there are things that could be better. I will likely revisit this project to make improvements in the future.
